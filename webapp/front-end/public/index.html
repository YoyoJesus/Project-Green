What is this project?


The goal of our project is to incentivize the recylcing of appropriate items using machine learning. By reading what types of waste is disposed of, we could potentially identify the exact types of waste items. Doing so would allow us to keep track of the items recycled and use that information to encourage further recycling. In our case, we would allocate points for each recycled item. Which could then be used to redeem coupons or other types of rewards for the user.
